Chapter 1: Cloud Concepts

Q1. A company wants to stop guessing their infrastructure capacity and avoid paying for idle servers. Which AWS cloud advantage directly addresses this? A) Benefit from massive economies of scale B) Stop spending money on data centers C) Stop guessing capacity D) Trade capital expense for variable expense

Click to View Answer & Explanation Correct Answer: C
Explanation: The cloud allows you to scale up or down automatically. You no longer need to guess how many servers you will need for peak times, thereby avoiding paying for idle servers.

Q2. Which deployment model relies entirely on third-party infrastructure and services accessed over the internet? A) On-premises B) Hybrid Cloud C) Private Cloud D) Public Cloud

Click to View Answer & Explanation Correct Answer: D
Explanation: Public Cloud utilizes third-party providers (like AWS) to deliver computing resources over the internet.

Chapter 2: Global Infrastructure

Q3. To achieve the highest availability for an application within a single AWS Region, how should it be deployed? A) Across multiple Edge Locations B) Local Zones C) Across multiple Availability Zones (AZs) D) Across multiple AWS Regions

Click to View Answer & Explanation Correct Answer: C
Explanation: Deploying across multiple Availability Zones within the same Region ensures fault tolerance and high availability if one data center goes offline.

Chapter 3: Core Services (Compute, Storage, Database)

Q4. Which AWS service provides a fully managed relational database with support for MySQL and PostgreSQL, offering enterprise-grade performance? A) Amazon RDS B) Amazon DynamoDB C) Amazon Aurora D) Amazon Redshift

Click to View Answer & Explanation Correct Answer: C
Explanation: Amazon Aurora is a proprietary, fully managed relational database engine compatible with MySQL and PostgreSQL, offering superior performance and availability compared to standard RDS.

Q5. A developer needs to run a short script that takes 5 seconds to execute, triggered by an S3 upload. Which compute service is the most cost-effective? A) Amazon EC2 B) AWS Lambda C) Amazon ECS D) AWS Elastic Beanstalk

Click to View Answer & Explanation Correct Answer: B
Explanation: AWS Lambda is serverless compute that runs code in response to events and charges you only for the exact milliseconds the code runs.

Chapter 4: Security and Compliance

Q6. Under the AWS Shared Responsibility Model, which of the following is the customer's responsibility? A) Patching the hypervisor B) Physical security of the data center C) Configuring IAM roles and policies D) Upgrading network infrastructure

Click to View Answer & Explanation Correct Answer: C
Explanation: Customers are responsible for "Security IN the Cloud," which includes managing Identity and Access Management (IAM), data encryption, and OS patching.

Chapter 5: Billing and Pricing

Q7. Which AWS tool allows a company to visualize their historical AWS spending and forecast costs for the next 12 months? A) AWS Pricing Calculator B) AWS Budgets C) AWS Cost Explorer D) AWS Trusted Advisor

Click to View Answer & Explanation Correct Answer: C
Explanation: Cost Explorer lets you analyze past and current spend, visualize trends, and forecast future costs.
