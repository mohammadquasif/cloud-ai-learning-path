AWS Cloud Practitioner Flashcards

Exam: AWS Certified Cloud Practitioner (CLF-C02)
Best Use: Fast recall after reading the course once
Exam Count: 65 total questions in 90 minutes. AWS states that 50 affect your score and 15 are unscored.

How To Study These Cards

Use these after each chapter and again in the final 24 hours. Say the answer before reading it. If you miss a card, go back to the related lesson or quick revision guide.

High-Yield Flashcards

Front Back
What benefit converts upfront hardware purchase into usage-based spend? Trade capital expense for variable expense.
What benefit avoids buying too much or too little capacity? Stop guessing capacity.
What benefit means resources can be provisioned in minutes? Increase speed and agility.
Which framework has Business, People, Governance, Platform, Security, and Operations perspectives? AWS Cloud Adoption Framework.
Which framework has Security, Reliability, Operational Excellence, Performance Efficiency, Cost Optimization, and Sustainability pillars? AWS Well-Architected Framework.
Multi-AZ mainly improves what? High availability inside one Region.
Multi-Region mainly supports what? Disaster recovery from regional failure.
Who manages physical data center security? AWS.
On EC2, who patches the guest operating system? Customer.
Best practice for root user? Enable MFA, avoid daily use, use IAM identities for daily work.
Best IAM identity for AWS services and temporary access? IAM role.
"Who did what, when, and from where?" AWS CloudTrail.
Metrics, logs, dashboards, alarms? Amazon CloudWatch.
Resource configuration history and compliance rules? AWS Config.
SQL injection and XSS protection? AWS WAF.
DDoS protection? AWS Shield or Shield Advanced.
Threat detection from logs and account activity? Amazon GuardDuty.
Centralized security findings? AWS Security Hub.
Sensitive data discovery in S3? Amazon Macie.
Compliance reports and agreements? AWS Artifact.
Virtual machines with OS control? Amazon EC2.
Event-driven code without servers? AWS Lambda.
Containers without EC2 host management? AWS Fargate.
Upload code and AWS provisions app infrastructure? AWS Elastic Beanstalk.
Short-term unpredictable compute? On-Demand Instances.
Interruptible lowest-cost EC2? Spot Instances.
Predictable 1-year or 3-year compute discount? Reserved Instances or Savings Plans.
Object storage for files, backups, and static sites? Amazon S3.
Unknown S3 access pattern? S3 Intelligent-Tiering.
Rare archive with millisecond retrieval? S3 Glacier Instant Retrieval.
Lowest-cost long-term archive with hours retrieval? S3 Glacier Deep Archive.
Block storage for one EC2 instance? Amazon EBS.
Shared Linux file system? Amazon EFS.
Windows SMB file share? Amazon FSx for Windows File Server.
Managed relational database? Amazon RDS or Aurora.
Serverless NoSQL with single-digit millisecond latency? Amazon DynamoDB.
Data warehouse and BI analytics? Amazon Redshift.
Graph database? Amazon Neptune.
In-memory cache? Amazon ElastiCache.
CDN and edge caching? Amazon CloudFront.
Dedicated private connection to AWS? AWS Direct Connect.
Instance-level firewall? Security group.
Subnet-level stateless firewall? Network ACL.
Queue for decoupling? Amazon SQS.
Pub/sub notifications? Amazon SNS.
Event routing and schedules? Amazon EventBridge.
Estimate cost before building? AWS Pricing Calculator.
Historical spend and forecast? AWS Cost Explorer.
Cost threshold alerts? AWS Budgets.
Rightsizing recommendations? AWS Compute Optimizer.
Consolidated billing? AWS Organizations.
Technical Account Manager? Enterprise Support.
Approved products for teams? AWS Service Catalog.
Third-party software and AMIs? AWS Marketplace.
